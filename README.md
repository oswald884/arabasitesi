Proje: Araba Kiralama ve Satış Sitesi

Kullanıcı Giriş kısmına kayıt olabilirsiniz.

Kullanıcı Kısmından yapılabilecek işlemler :
1) Yeni kayıt yapılabilir
2) Kayıtlı hesaba giriş yapılabilir.
3) Oturum kapatılabilir
4) Admin kısmına mesaj gönderilebilir
5) Kiralama talebi gönderilebilir (Filtreleme mevcut)
6) Satış talebi gönderilebilir (Filtreleme mevcut)

Admin Kısmından yapılabilecek işlemler :
-> Ana Sayfada
1) Bekleyen kiralama siparişleri görüntülenebilir. Sipariş onaylanabilir veya reddedilebilir.
2) Önceden onaylanan kiralama siparişleri görüntülenebilir.
3) Satın alım talepleri görüntülenebilir. Araç satıldı işaretlenirse
araç stok tablosundan silinir ve satılanlar tablosuna aktarılır.
4) Okunmamış mesajlar görüntülenebilir.

-> Profil Sayfasında
1) Açık olan oturumun bilgileri görüntülenir.
2) Bütün bilgiler güncellenebilir.

-> Marka Ekle Sayfasında
1) Yeni Marka eklenebilir.

-> Markalar Sayfasında
1) Mevcut markalar görüntülenebilir.
2) Seçilen marka silinebilir. 
(Markayla ilgili modeller ve modeller ile ilgili stoktakiler de silinir.)
3) Marka bilgileri düzenle seçeneğinden güncellenebilir.

-> Model Ekle Sayfasında
1) Yeni Model eklenebilir.

-> Modeller Sayfasında
1) Mevcut modeller görüntülenebilir.
2) Seçilen model silinebilir. (Model ile ilgili stoktakiler de silinir.)
3) Model bilgileri düzenle seçeneğinden güncellenebilir.

->Stoğa Araç Ekle Sayfasında
1) Yeni araç eklenebilir.

-> Stoktakiler Sayfasında
1) Mevcut araçlar görüntülenebilir.
2) İstenen araç silinebilir.
3) Araç bilgileri düzenle seçeneğinden güncellenebilir. (Aracın aktif/pasif olması veya kiralık/satılık olması da dahil)

-> Admin Ekle Sayfasında
1) Yeni Admin eklenebilir.

-> İçerik Düzenle Sayfasında
1) Kullanıcı kısmındaki ana sayfanın içeriği değiştirilebilir. (Resimler, başlıklar, paragraflar)
